# Photo Gallery

A simple, mobile-friendly photo gallery web app. Paste image URLs to add photos, view them in a beautiful grid, and browse through them with a lightbox viewer.

## Features

✨ **Easy to Use**
- Paste image URLs to add photos instantly
- No backend or authentication needed
- Works completely in your browser

📱 **Mobile Optimized**
- Responsive design works on phones, tablets, and desktops
- Touch-friendly controls
- Perfect for adding photos on the go

🖼️ **Gallery Features**
- Grid layout that adapts to screen size
- Lightbox viewer for full-size image viewing
- Arrow navigation to browse through photos
- Delete button to remove photos
- Photos saved locally in browser storage

## How to Use

1. Open `index.html` in your browser
2. Paste an image URL in the input field
3. Click "+ Add Photo" or press Enter
4. Click any photo to open the lightbox viewer
5. Use arrow buttons to navigate between photos
6. Click the × button to close the lightbox
7. Hover over a photo and click × to delete it

## Local Storage

Your photos are automatically saved in your browser's local storage. They'll persist even after you close and reopen the page (until you clear browser data).

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (to load images from URLs)

## Tips

- Use image hosting services like:
  - Imgur
  - Pixabay
  - Pexels
  - Unsplash
  - Google Photos (public links)
  
- Make sure the image URL points directly to the image file (not a web page)
